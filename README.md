# Jekyll Blocker

Jekyll plugin that builds pages from sitemap, layouts and section blocks

## Installation

Add this line to your site's Gemfile:

```ruby
gem 'jekyll-blocker'
```

And then add this line to your site's `_config.yml`:

```yml
plugins:
  - jekyll-blocker
```
